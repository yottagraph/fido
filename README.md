# Fido Fetch Project

Fido plays fetch!

This template repo is intended to be modified to work on a new data source. The expectation is that the template will be created with an initial data model (schema and data dictionary), as well as an initial exploration of the data source. This repo provides the actual code for pulling the data from the source and writing the data files to a GCS bucket. It is intended to be run as a cloud run job.
